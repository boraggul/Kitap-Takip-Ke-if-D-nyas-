 Kitap Takip ve Keşif Dünyası

Bu proje, okunan veya okunmak istenen kitapları düzenli bir şekilde listelemek, takip etmek ve yeni kitaplar keşfetmek amacıyla geliştirilmiş web tabanlı bir kitap yönetim uygulamasıdır.

 Özellikler
- **Kitap Yönetimi:** Yeni kitaplar ekleyebilir, mevcut kitapları listeleyebilirsiniz.
- **Dinamik Veri:** Kitap bilgileri arka planda JSON formatında güvenli bir şekilde saklanır.
- **Kullanıcı Dostu Arayüz:** Sade ve anlaşılır HTML şablonları ile kolay kullanım sağlar.

 Kullanılan Teknolojiler
- **Backend:** Python / Flask
- **Frontend:** HTML5, CSS3
- **Veri Saklama:** JSON (`kitaplar.json`)

  Kurulum ve Çalıştırma

Projeyi yerelde çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1. Bu depoyu bilgisayarınıza indirin veya klonlayın.
2. Gerekli bağımlılıkların yüklü olduğundan emin olun (Flask vb.).
3. Ana uygulama dosyasını çalıştırın:
   ```bash
   python app.py
